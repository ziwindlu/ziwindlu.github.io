docker使用

windows中使用docker

安装

Windows Docker 安装 | 菜鸟教程 (runoob.com)

使用wsl连接docker

使用WSL连接Docker Server(Windows/Linux) - CieloSun - 博客园 (cnblogs.com)

镜像操作命令

search

搜索镜像

docker search ubuntu

pull

拉取镜像

docker pull ubuntu:tag

images

查看拉取的镜像

rmi

删除镜像

docker rmi imageName

容器操作命令

run

启动容器

docker run options <image>[:tag] <cmd>

如果tag不存在时，默认使用latest

常用参数

options means example
-i 在容器内指定一个终端或伪终端 <++>
-t 允许与容器交互 <++>
-d 后台运行 <++>
-P 将容器内部使用的网络端口随机映射到主机上 <++>
-p 映射端口 -p localPort:dockerPort
-v 映射数据卷 -v localDir/volumeName:dockerDir

ps

查看容器状态

常用参数

options means
-a 列出所有容器，包括隐藏的

stop

停止某容器

start

开启某已停止的容器

restart

重启某容器

exec

执行命令

docker exec options container cmd

import

导入

docker import [OPTIONS] file|URL|- [REPOSITORY[:TAG]]

export

导出

docker export [OPTIONS] CONTAINER

options means
-o 将内容输出到指定文件

logs

查看容器日志

docker logs container

port

查看容器端口映射

docker port container

rm

移除容器

docker rm container

inspect

docker inspect container

查看docker容器信息

Docker格式化输出命令:"docker inspect –format" 学习笔记 - 散尽浮华 - 博客园

数据卷

数据卷包含创建、修改、删除、查看、详情、删除未使用(prune)

创建的数据卷在dockerRoot/volume下

一个简单的使用

docker volume create mysql-data

创建一个数据卷

以上命令创建了一个mysql容器并将数据卷mysql-data映射到了容器中/var/lib/mysql/

网络

网络包含创建、连接、断开连接、删除、查看、详情、删除未使用(prune)

小技巧

docker设置为开机自启动

docker update --restart=always id

docker存储目录迁移

将/var/lib/docker迁移
