chocolatey 获得APT似的快感

一个包管理工具，简化安装步骤。避免安装到盗版软件

官网

Chocolatey Software | Chocolatey - The package manager for Windows

安装chocolatey

参考Windows使用Chocolatey安装软件 - 知乎 (zhihu.com)

可能会遇到的问题

  • windows禁止运行外部脚本(Windows上PowerShell默认禁止运行 *.ps1 脚本原因与解决方法 - 简书 (jianshu.com))

  • 安装chocolatey速度慢(chocolatey 在国内使用_古承风的博客-CSDN博客_choco 国内源)

我能装什么

官方的库中有许多软件，在Chocolatey Software | Packages这里搜索即可

choco和apt相似，你可以使用其他库安装官方库中没有的软件

我要怎么装

Win-X选择管理员运行powershell，在powershell中输入以下命令即可安装

choco install packageName

wox 获得dmenue的体验

一个功能强大的启动器，你甚至可以将他当作翻译软件。丰富的插件，简化windows中的操作

官网

wox

安装wox

当然是使用choco安装了

choco install wox

使用

仅需将关键词键入框内，和关键词相关的内容就会出现

一些快捷键

keybinds means
<C-H> 历史记录
<C-N/P> 上下移动
<C-J/K> 上下移动

插件

wox内置了很多插件，可以选择性开启，默认全部开启。

开启or关闭插件

搜索框内键入settings以编辑设置。设置->插件

Plugin (wox.one)在这里找

打开某个插件的页面后，插件会告诉你怎么安装，怎么使用

推荐

  • 有道翻译

  • Clipboard History快捷访问历史记录

  • IP Address查询本机ip

  • 随手记

  • Todos简易的todo

  • 有道智云翻译支持发音

  • Wox.Plugin.Githubgithub搜索

  • Chocolatey你甚至可以使用choco

  • Alias别名

个性化配置

将win-S绑定为启动的快捷键，设置->热键->wox激活热键

将win-T绑定为有道翻译的快捷键，设置->热键->添加热键触发关键字填入yd 即可，注意这里的空格，不然每次使用时需要另外添加空格

配置时会遇到的问题

windows热键被占用Windows如何禁用部分特定或全部键盘快捷键 - 系统极客 (sysgeek.cn)

我的别名方案

在某处复制并修改快捷方式的名字达到别名的功能

everything 比find不知道高到哪里去了

CClose 简化鼠标操作逻辑键盘党的福音

这款软件最好用的功能是将某个窗口置顶

官网

CClose (chaohershi.github.io)

优化

windows中默认开启Aero Shake功能，即在摇晃一个窗口时，windows会将当前桌面上的所有窗口最小化。这会导致CClose的窗口置顶功能失效

如果你是一个手残用户，请禁用这个功能

How to disable Aero Shake on windows10

How to Enable or Disable Aero Shake on Windows 10 - Technastic

How to Disable Aero Shake on Windows 10 (groovypost.com)

注意，彻底关闭贴靠窗口会解决此问题，但windows将无法使用某些功能，建议修改策略组中相关内容。

关闭贴靠窗口无法使用的功能:快速调整窗口大小(win-UP/DOWN/LEFT/RIGHT)

谨防非UWP软件

UWP软件的好处

UWP软件的权限小，对用户隐私保护较好，不会出现某些软件扫描浏览记录的情况。会经过微软审核，面向全球。
