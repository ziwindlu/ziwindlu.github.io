kitty使用

发表于2023-08-17|更新于2023-08-18|折腾日记工具使用自有配置快捷键快查
|字数总计:610|阅读时长:2分钟|阅读量:

一个高度可定制化的终端

以下快捷键均为自定义的，大部分只是将ctrl+shift替换为了alt

本人的快捷键定义
- 系统使用win键、Fn+F*
- 终端使用alt键、F*
- 软件使用除了win以外的所有，需要在终端运行的软件尽量避免alt的使用
- 尽量避免alt+win的操作

通用的

定义modkitty_mod alt

c-s-c,c-s-v为复制粘贴

快捷键

屏幕滚动

shortcut action
mod+up 向上一行
mod+down 向下一行
mod+page_up>u 向上一页
mod+page_down>d 向下一页
mod+z 到上一个命令输入的地方
mod+x 到下一个要输入命令的地方
mod+home 跳转到最上面
mod+end 跳转到最下面
mod+h 使用less浏览历史(已经使用alias vimhis代替)
mod+g 使用less浏览上一个命令的输出

tab

shortcut action
mod-t 新建tab
mod+q 关闭tab
mod-right/left 移动tab
c-s-,/. 将tab往前往后移动
mod+shift-t 命名tab
mod+0-9 跳转到对应的tab

window

shortcut action
mod+enter 新建一个窗口
mod+shift+enter 新建一个窗口并自动进入工作目录
mod+w 关闭窗口
mod+r 调整窗口大小(比较反人类)
mod+[/] 窗口间移动(emmm,勉强能用)已经调整为hjkl
mod+h/l/j/k 窗口移动like i3
mod+n 切换布局，kitty默认将所有布局都开启了详情请见kitty layouts
mod+shift+up/down/left/right 调整窗口大小

配置相关

kitty使用配置文件进行配置,这个通常用不到

C-S-F2打开配置文件

C-S-F5重新加载配置

C-S-F6查看现在的配置

用不到

remote

hello world

在同一个kitty进程中kitty可以向另一个kitty窗口发送指令之类的

来自官网的一个实例

Hello World

这将会开启一个新的窗口，我理解为kitty客户端，并且运行了cat

这会让kitty去找寻cmdline运行了cat的kitty远程客户端，并发送Hello, World，cat命令接收到Hello World将会输出

同样的你可以使用管道符将ls的输出放到Output中

如果你发现你的kitty并不能使用远程，可能是配置问题，可去官网查询
