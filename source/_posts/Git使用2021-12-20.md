### 仓库创建相关

`git clone`克隆仓库  
`git init`新建一个仓库  

### 工作相关

`git add`添加文件  
`git commit`提交更改  
`git status`显出工作区状态  
`git rm/mv`文件删除，移动  
`git log`查看提交日志  
`git reset logNumb`恢复到**logNumb**并清除提交记录  
`git revert logNumb`恢复到**logNumb**不清除提交记录

## 远程仓库

`git remote`远程操作  
`git fetch`获取远程仓库更新  
`git pull`从远程仓库与本地合并  
`git push`推送到远程仓库
`-f`强制推送
