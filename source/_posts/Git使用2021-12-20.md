## git 中的概念

## gi command

### 仓库创建相关

`git clone`  克隆仓库  
`git init`  新建一个仓库  

### 工作相关

`git add`  添加文件  
`git commit`  提交更改  
`git status`  显出工作区状态  
`git rm/mv`  文件删除，移动  

### 远程仓库

`git remote`  远程操作  
`git fetch`  
`git pull`  
`git push`
