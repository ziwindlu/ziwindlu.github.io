archlinux下samba服务搭建

发表于2024-04-11|更新于2024-04-16
|字数总计:283|阅读时长:1分钟|阅读量:

安装所需安装包

pacman -S samba

修改配置文件

samba没有提供配置文件，需要手动创建。

创建/etc/samba/smb.conf文件，内容如下

更多配置项samba.org

管理smb用户

samba需要linux用户才能使用

修改samba密码

smbpasswd -a mysmb

samba管理工具

pdbedit

创建用户

pdbedit -a username

注意这个username是要已经在linux用户中存在的

删除用户

pdbedit -x username

查看用户

pdbedit -L -v

启动服务

systemctl start smb.service nmb.service

开机自启动

systemctl enable smb.service nmb.service

使smb服务可被发现

使smb被linux、mac发现

安装软件包pacman -S avahi

启动服务systemctl start avahi-daemon.service

使smb被windows发现

安装软件包yay -S wsdd

启动服务systemctl start wsdd.service

windows下连接

网络->映射网络驱动器->文件夹框内填入\\ipaddres\homes->输入账号密码连接
